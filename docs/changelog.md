## V1.0.0 (2026-04-26)
### 新增
- 多集群管理：K3s集群注册、状态监控、心跳检测、在线/离线状态实时展示
- 应用市场：应用模板管理，Helm Chart配置，版本控制
- 声明式部署：期望状态与实际状态对比同步，配置漂移检测与自动修复
- WebSocket隧道：管控面与集群安全隧道，K8s API代理转发（白名单模式）
- 组织管理：多层级组织架构（部门/项目/团队）
- 监控告警：内置5种告警规则（集群离线/CPU过高/内存过高/配置漂移/License过期），自动评估引擎
- 配置中心：配置模板管理，按组织维度管理Helm Values
- License管理：许可证激活和配额管理
- 用户管理：多角色（admin/viewer），JWT认证，随机初始密码
- 审计日志：自动记录用户操作
- 仪表盘：多维度统计概览，License配额可视化
- 批量部署：支持选择多集群同时部署同一应用
- 集群详情：标签管理、指标趋势、安装脚本/注册YAML展示
- Agent安装脚本：一键安装K3s+Agent

### 安全
- JWT认证 + Agent Token双重认证
- K8s API代理白名单模式（仅允许安全GET请求）
- SQL注入防护（参数化查询）
- CORS Origin可配置
- WebSocket CheckOrigin校验
- 删除操作二次确认
- bcrypt密码加密
